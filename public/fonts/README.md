# Fonts

Add **Le Jour Script** here for the wedding site titles:

- **File name:** `LeJourScript.woff2` (or `LeJourScript.woff`)
- **Where to get it:** Purchase or download from [MyFonts](https://www.myfonts.com/collections/le-jour-font-din-studio) or other font retailers. Check the license for web use.

If you have the font in another format (e.g. `.ttf`), convert it to WOFF2 using a tool like [CloudConvert](https://cloudconvert.com/ttf-to-woff2) and save as `LeJourScript.woff2`.
