# Wedding Website

A simple, elegant wedding website built with Next.js 14 (App Router) and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

- **RSVP:** Replace `rsvp@ourwedding.com` with your email or link to a form/RSVP service.

## Build

```bash
npm run build
npm start
```
